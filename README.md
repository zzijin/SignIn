# 小组项目:SignIn(签到打卡)


#### 介绍
SignIn项目的Android客户端，使用JAVA编写


#### 项目链接:
C#服务器：https://gitee.com/zzijin/sign-in-server


#### 技术要点


#### 功能实现

一、用户操作：
   1.用户登录/注册
   2.用户修改个人信息

二、打卡活动：
   1.用户创新新的打卡活动
   2.活动发起者修改管理活动信息
   3.用户查看参与的活动
   4.用户通过搜索活动ID加入活动
   5.首页推荐活动
   6.搜索相关主题活动

三、定位辅助：
   1.发布活动时通过自我定位/文字选址/地图点击确认打卡定位位置
   2.去往参与活动时可使用地图导航路线
   3.打卡时间内通过地图定位完成打卡
